# AngularJS with Webpack

AngularJS 1.8, Webpack 4, Babel 7, ES6 modules, Karma, HMR (Hot Module Reload) enabled

## Features

- Latest AngularJS version
- Webpack 4
- Babel 7
- CoreJS
- SASS support
- Source mapping
- ngAnnotate support
- HMR (Hot Module Reload) support via Babel plugin
- ESLint
- Husky + lint-staged + Prettier

## Getting Started

```
npm install
```

## Running the project

For development

```
npm start
```

For production

```
npm build
```

For testing

```
npm test
```
